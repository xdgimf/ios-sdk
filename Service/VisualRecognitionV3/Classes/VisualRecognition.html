VisualRecognition

public class VisualRecognition

The IBM Watson Visual Recognition service uses deep learning algorithms to analyze images, identify scenes, objects, faces, text, and other content, and return keywords that provide information about that content.

    The name of the new classifier.

    An array of classes, each with a name and a zip archive file of images that prominently depict the visual subject of the class. Each class requires a minimum of 10 images. If you specify multiple classes, the system will learn to classify each category.

    A zip archive file of images that prominently depict the visual subject of of the classes being trained. Must contain a minimum of 10 images.
